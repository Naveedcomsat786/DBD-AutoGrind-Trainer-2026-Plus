## ⚙️ Key Features

The trainer includes several modules designed to simplify game mechanics.

### Bloodpoint Automation
The bloodpoint bot gathers currency while you step away from the keyboard. It performs repetitive actions and interactions to maximize your earnings per hour. 

### ESP Wallhack
This visual overlay marks player locations through walls. It draws thin outlines around survivors or killers to help you navigate the map. You can toggle this feature at any time using the hotkey provided in the settings menu.

### Aimbot Assistance
The aimbot adjust your crosshair position toward targets when you move the mouse. It relies on internal distance calculations to keep your shots accurate. You can adjust the speed and smoothness of this movement in the trainer dashboard.

### Perk and Skin Unlocker
This module modifies your local configuration to grant access to items. It scans your inventory and applies the digital keys required to equip rare skins or maxed-out perks.

## 🎛 Configuration Settings

Launch the settings menu by pressing the F10 key while the trainer is running. You will see several sliders and checkboxes:

* Draw Distance: Control how far the wallhack renders entities.
* Automation Speed: Regulate how fast the bloodpoint bot interacts with the game.
* Refresh Rate: Set how often the overlay updates the screen positions.
* Save Settings: Press F12 to lock your changes for the next session.

## ❓ Troubleshooting

If you encounter issues, review these common solutions before you try anything else.

### Tool does not launch
Verify that you run the file as an administrator. Windows security features may block external tools unless you grant them high-level permissions. Ensure you have the latest .NET Framework installed on your system. 

### Overlay does not appear
Check that the game runs in Borderless Windowed mode. Fullscreen mode can prevent the overlay from rendering on top of the game window. 

### Performance drop
Lower the refresh rate in the settings menu. High refresh rates require more system resources. If your frame rate remains low, close unnecessary background applications like web browsers or video players.
